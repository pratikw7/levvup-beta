Help

How to use:

Accept notifications when prompted to receive them

Create a new task by pressing the '+' icon on the home page

LEVVUP helps you track your daily tasks. You can mark them as complete once you've completed them, just like a To-do list!

LEVVUP believes in daily progress and is built to help you become consistent at leveling up in real life!

Any large task can be broken down into small daily milestones.(Eg: If final goal is 5kg weight loss in a month then add 30mins cardio as your task daily)

To save your progress press the save button (Cassette icon)

You have 12hrs until the today's tasks auto expires and gets added to your history! MM/DD/YYYY is the date format

Remember: Every friend you can see your tasks and its completion status

Add friends to make yourself more accountable!

You must add friends by sending a friend request to the email ID which they used to register on LEVVUP.

Adding friends helps you to create an environment for the both you to level up daily!


Other information:

Every task completed adds 10 XP and every task missed for the day deducts 5 XP

LEVVUP progress bar denotes your progress towards levelling up

Task progress bar denotes how many daily tasks you have completed
